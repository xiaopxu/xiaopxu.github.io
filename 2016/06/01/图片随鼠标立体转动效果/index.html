图片随鼠标立体转动效果

  • 效果Demo

前几天看到锤子手机官网首页上面的轮播图特效，发现轮播图会随着鼠标的移动进行3d旋转，效果非常棒；所以自己也尝试着写了一个类似的效果

锤子手机原网站效果点我传送


实现思路：

  1. 用event事件获取鼠标位置
  2. 将鼠标移动的位置除以图片长度的一半，得出鼠标相对于半图片的移动比例
  3. 设定onmousemove事件，让图片的3d旋转角度根据鼠标的移动比例变化
  4. 将各参照层Z轴位置错开；并给图片设置适当的阴影让视觉效果更佳棒

先上代码看看

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Document</title>

    //css 部分
    <style>
        *{
            margin: 0;
            padding: 0;
        }
        #box{
            width: 1000px;
            height: 600px;
            position: relative;
            margin: 200px auto;
            transform-style: preserve-3d;   //这个样式是将3d效果开启，需要加给进行3d变换元素的父级
            background-color: #ccc;
        }
        img{
            width: 600px;
            height: 400px;
            position: absolute;
            left: 200px;
            top: 100px;
            display: block;
            z-index: 10;
            transition: all 0.3s;
        }
        h1{
            font-family: arial;
            position: absolute;
            right: 500px;
            bottom: 200px;
            color: red;
            font-size: 50px;
            z-index: 15;
        }
        h2{
            font-family: arial;
            position: absolute;
            font-family: arial;
            right: 400px;
            bottom: 200px;
            color: #fff;
            z-index: 15;
        }
    </style>
</head>

//html部分
<body>
    <div id="box">
        <h1>立体效果</h1>
        <h2>平面效果</h2>
        <img src="http://rescdn.qqmail.com/dyimg/20140617/789233B10749.jpg" alt="">
    </div>
</body>

//js部分
<script>

    var box = document.getElementById('box');
    var img = document.getElementsByTagName('img')[0];
    var text = document.getElementsByTagName('h1')[0];
    var text2 = document.getElementsByTagName('h2')[0];


    box.onmouseenter = function () {
        box.onmousemove = function (e) {

            //获取event属性
            var ev = e||window.event;     

            //计算鼠标相对于div中的位置
            var L = ev.clientX - box.offsetLeft;   
            var T = ev.clientY - box.offsetTop;

            //设置鼠标可以移动的范围
            var minL = box.offsetLeft;
            var maxL = box.offsetLeft + box.offsetWidth;
            var minT = box.offsetTop;
            var maxT = box.offsetTop + box.offsetHeight;

            //计算图片横纵方向一半位置的坐标
            var halfL = (maxL - minL) / 2;
            var halfT = (maxT - minT) / 2;

            //设置图片变化最大值
            var maxAngle = 20;        //图片最大角度
            var maxTextAngle = 20;    //文字最大角度
            var maxShadow = 10;       //阴影最大角度

            //计算鼠标相对于半图片的移动位置比例
            var scaleX = (L - halfL) / halfL;  //左边半边是负值，右半边是正值；即从左到右是 -30度→ 0度→ +30度
            var scaleY = (T - halfT) / halfT;  //上边半边是负值，下半边是正值；即从上到下是 -30度→ 0度→ +30度

            //图片的3d角度变换及Z轴位置，Z轴错开易于体现3d效果
            //transform: rotate(角度) translate(位移)，分别可以用xyz轴来区分
            img.style.transform = 'translateZ(500px) rotateY(' + scaleX * maxAngle + 'deg) rotateX(' + scaleY * (-maxAngle) + 'deg) rotateZ(0deg)';

            //文字3d角度变换设置及Z轴位置
            text.style.transform = 'translateZ(10000px) rotateY(' + scaleX * maxTextAngle + 'deg) rotateX(' + scaleY * (-maxTextAngle) + 'deg) rotateZ(0deg)';

            //静态文字Z轴位置设置
            text2.style.transform = 'translateZ(5000px)';

            //图片阴影设置，这个也是根据图片转动来动态变化的
            //让抬起的方向阴影深，落下的方向阴影浅，就能很容易的出现视觉上的角度变换
            //box-shadow: 
            //第一个是X方向阴影偏移，右正左负
            //第二个是Y方向阴影偏移，下正上负
            //第三个是阴影扩散范围
            //最后一个是阴影颜色
            img.style.boxShadow = scaleX * (-maxShadow) + 'px ' + scaleY * (-maxShadow) + 'px 20px black';
        }
    }
</script>
</html>
