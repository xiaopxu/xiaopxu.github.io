js数据类型查看typeof方式的优化

  • JavaScript

JavaScript中有两种不同数据类型的值，分别是基本数据类型与引用数据类型

  • 基本数据类型包含5类，分别是：Number、String、Boolean、Null、Undefined
  • 引用数据类型包含剩下的Array、Date、Math、RegExp、Object、Function等

如果我们要查看某个变量的数据类型，可以使用typeof操作符

var num = 123;
var str = 'abc';
var bol = true;
var n = null;
var un = undefined;
var arr = [];
var obj = {};
var re = /\d+/;
var m = Math;

console.log( typeof num );     //number
console.log( typeof str );  //string
console.log( typeof bol );  //boolean
console.log( typeof n );  //object
console.log( typeof un );    //undefined
console.log( typeof arr );  //object
console.log( typeof obj );  //object
console.log( typeof re );  //object
console.log( typeof m );  //object

可以看出：typeof操作符对于简单数据类型的识别比较优秀，但是引用数据类型基本都是返回的object，导致我们并不能知道具体类型

我们可以将这种判定方法进行一定的优化，实现对于引用数据类型比较准确地判断

实现方法：
  • Object对象的prototype的toString()方法会返回一个包含对象本身类型的字符串
  • 将字符串中需要的部分截取出来

toString()方法使用

console.log(Object.prototype.toString())  //[object Object]

这里第二个打印值就是我们需要的类型

因为这里的this指向Object本身，所以返回的是Object

我们可以在使用方法时用call方法改变this的指向，变成我们需要判定的变量即可

var arr = [];
console.log(Object.prototype.toString.call(arr))  //[object Array]

现在我们用字符串操作符slice将需要的部分截取出来，再将字母全部变为小写就得到需要的返回值了

var arr = [];
console.log(Object.prototype.toString.call(arr))  //[object Array]
console.log(Object.prototype.toString.call(arr).slice(8,-1).toLowerCase())  //array
所以最后的实现方式如下：
//封装成type函数调用
function type(obj){
    var o = {};
    return o.toString.call(obj).slice(8,-1).toLowerCase();
}

//测试如下
var arr = [];
var obj = {};
var re = /\d+/;
var m = Math;

console.log(
    type( arr ),     //array
    type( obj ),      //object
    type( re ),      //regexp
    type( m ),          //math
    type( window ),   //window
    type( document ),  //htmldocument
    type( function(){} ),  //function
    type( 'abc' ),      //string
    type( null ),       //null
    type( undefined )    //undefiend
);

这样就可以得到我们要的效果，这些返回会方便我们在if语句中进行条件判断
