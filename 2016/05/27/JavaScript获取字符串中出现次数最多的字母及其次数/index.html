JavaScript获取字符串中出现次数最多的字母及其次数

  • JavaScript

如何获得字符串中出现次数最多的字母及其长度？

思路：

1）循环字符串，每次截取一个字母

2）剔除字符串中全部的该字母，并得到新的字符串

3）将原始字符串长度减去行字符串长度，得到截取字母的长度

4）判断该长度是否现有最长，若是，则储存，并将对应的字符储存

5）循环完成整个字符串，输出储存的结果

代码如下：

var str = "aabccddaaaccddddddd";
var maxLen = 0;  //用来储存最大长度
var letter = "";  //用来储存最大长度对应字母
for (var i = 0; i < str.length; i++) {
    var getStr = str.substr(i,1);  //每次循环将str中对应位置的字母截取出来
    var arr = str.split(getStr)  //用截取出的字母分割str，并生成一个新数组，新生成的数组中所有截取字母都是逗号，连续的截取字母中间会出现""
    var newStr = arr.join("");  //用多出的""作为分隔标志将数组还原为newStr，到此剔除循环截取的字母
    var deltaLen = str.length - newStr.length; //用原字符串长度减去新字符串长度（剔除截取字母的）就得到截取字母个数
    if (deltaLen > maxLen) {
        maxLen = deltaLen   //如果字母是当前最长的，将字母个数给len
        letter = str[i];  //将该字母给letter
    }
}
console.log(letter + "的长度最长，是" + maxLen + "个")

说明：

我用来剔除全部截取的字母的方法略复杂，思路如下：

1）将原字符串str用截取字母作为分隔符，生成数组arr

2）用空字符串””作为分隔符，再将arr转回新的数组newStr，以去除多出的空字符串””

3）得到的newStr就是我们想要的剔除了截取字母的字符串

最简便的方法是博客园看到的：

利用的字符串的 replace 方法将符合正则表达式（第一个字符）替代为空

eval("str = str.replace(/"+getStr+"/g,'')");
