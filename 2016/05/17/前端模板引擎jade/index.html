前端模板引擎jade学习（一）

  • jade

安装

$ npm install gulp-jade --save

调用

  1. 在gulpfile.js中调用
var jade = require('gulp-jade')
  1. 新建jade任务
gulp.task('jade',function){
    gulp.src('view/*.jade')  //导入外部jade文件
    .pipe(jade())            //执行jade()方法
    .pipe(gulp.dest('desk/html/'))    //导出在新文件夹中
}

  1. 终端执行编译，生成html文件（压缩形式）

    $ gulp jade

    语法

  2. html：<开始标签>+内容+</结束标签>；jade：开始标签+空格+内容
<h1 id="title">我是h1</h1>    //html
h1#title 我是h1        //jade
  1. 嵌套结构使用Tab缩
body
   h1#title 我是h1
  1. 标签内属性使用括号引入
<meta charset="UTF-8">  //html
meta(charset='UTf-8')   //jade
  1. id名：#id；类名：.class
<h1 id="title"></h1>        //html
<div class="box"></div>        //html
h1#title    //jade
div.box        //jade

  1. 单行注释：//

    //我是注释

  2. 模块引入：

    include head.jade //会在此处引入已经写好的其他jade模块内容

完整举例：

//html语言        
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Document</title>
</head>
<body>
    <h1 id="title">我是h1</h1>
    <div class="box">
        <p>我是p</p>
        <ul>
            <li>我是<span>一个</span>li</li>
            <li></li>
            <li></li>
        </ul>
    </div>
</body>
</html>


//jade言
doctype
html(lang="en")
    head
        meta(charset='UTf-8')
        title Document
    body
        h1#title 我是h1
        div.box
            p 我是p
            ul
                li 我是        //此处嵌套写法可以关注一下
                    span 一个
                    | li    //竖杠来延续在li之中的span后追加内容
                li
                li

//编译后
