JavaScript获取字符串中出现次数最多的字母及其次数

如何获得字符串中出现次数最多的字母及其长度？

思路：

1）循环字符串，每次截取一个字母

2）剔除字符串中全部的该字母，并得到新的字符串

3）将原始字符串长度减去行字符串长度，得到截取字母的长度

4）判断该长度是否现有最长，若是，则储存，并将对应的字符储存

5）循环完成整个字符串，输出储存的结果

前端模板引擎jade学习（一）

安装

$ npm install gulp-jade --save

调用

  1. 在gulpfile.js中调用
var jade = require('gulp-jade')
  1. 新建jade任务
gulp.task('jade',function){
    gulp.src('view/*.jade')  //导入外部jade文件
    .pipe(jade())            //执行jade()方法
    .pipe(gulp.dest('desk/html/'))    //导出在新文件夹中
}

  1. 终端执行编译，生成html文件（压缩形式）

    $ gulp jade
